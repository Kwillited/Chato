# Chato 文档

欢迎查看 Chato 项目的官方文档。本目录包含了 Chato 应用的所有相关文档，包括架构设计、开发指南、用户指南等。

## 文档目录

### 1. 架构设计

- **[ARCHITECTURE.md](ARCHITECTURE.md)** - 详细描述了 Chato 应用的整体架构设计，包括前端架构、后端架构、数据存储设计等，包含架构图和核心技术栈说明。

### 2. 模块/服务详述

- **[MODULES.md](MODULES.md)** - 详细描述了 Chato 应用的各个模块和服务，包括其功能、架构、依赖关系和主要组件。

### 3. 开发与部署指南

- **[DEVELOPMENT_GUIDE.md](DEVELOPMENT_GUIDE.md)** - 提供给开发者的详细指南，包括环境搭建、开发流程、代码风格、测试、构建与部署等。

### 4. API 接口文档

- **[API.md](API.md)** - 详细描述了 Chato 应用的 API 接口，包括接口地址、请求方法、参数说明、响应格式等。

### 5. 协议与集成文档

- **[PROTOCOLS.md](PROTOCOLS.md)** - 详细描述了 Chato 应用使用的各种协议和集成方式，包括 API 协议、通信协议、第三方服务集成等。

### 6. 用户指南

- **[USER_GUIDE.md](USER_GUIDE.md)** - 面向最终用户的使用手册，包括应用安装、界面介绍、功能使用、故障排除等。

### 7. 贡献指南

- **CONTRIBUTING.md** - 如何为 Chato 项目贡献代码和文档的指南（待创建）。

### 8. 变更日志

- **CHANGELOG.md** - 记录 Chato 应用的版本变更历史（待创建）。

## 如何使用文档

1. **开发者**：建议先阅读 [DEVELOPMENT_GUIDE.md](DEVELOPMENT_GUIDE.md) 和 [ARCHITECTURE.md](ARCHITECTURE.md)，了解项目架构和开发流程。
2. **最终用户**：建议阅读 [USER_GUIDE.md](USER_GUIDE.md)，了解如何使用 Chato 应用的各项功能。
3. **贡献者**：建议阅读 [CONTRIBUTING.md]（待创建），了解如何为项目贡献代码和文档。
