# 用户指南

## 1. 应用概述

Chato 是一个基于 Tauri 2 和 Vue 3 开发的跨平台桌面应用，专注于提供强大的 AI 对话智能体功能，集成了 MCP（模型控制面板）和企业级 RAG（检索增强生成）能力。

## 2. 安装与启动

### 2.1 系统要求

- Windows 10/11 (64位)
- macOS 11+ (Intel/Apple Silicon)
- Linux (64位)

### 2.2 安装方法

1. 从官方网站下载对应操作系统的安装包
2. 双击安装包，按照向导完成安装
3. 安装完成后，在应用列表中找到 Chato 并启动

### 2.3 首次启动

首次启动应用时，您将看到欢迎界面，引导您完成初始设置：

1. 选择默认 AI 模型
2. 设置语言偏好
3. 配置通知选项

## 3. 用户界面

### 3.1 主界面布局

Chato 应用界面分为以下几个主要区域：

1. **顶部导航栏**：应用标题、菜单按钮、设置按钮
2. **左侧面板**：
   - 聊天历史记录
   - 新聊天按钮
3. **中央聊天区域**：
   - 聊天消息列表
   - 输入框
4. **右侧面板**：
   - 文件管理
   - MCP 控制面板
   - 其他功能面板

### 3.2 界面元素

#### 3.2.1 聊天区域

- **消息气泡**：显示用户和 AI 的对话
- **滚动到底部按钮**：快速滚动到最新消息
- **消息操作**：复制、删除消息等

#### 3.2.2 输入框

- **文本输入**：输入聊天内容
- **发送按钮**：发送消息
- **附件按钮**：上传文件
- **设置按钮**：调整聊天设置

#### 3.2.3 侧边栏

- **聊天历史**：查看和管理历史聊天
- **文件管理**：上传、查看和管理文档
- **MCP 面板**：管理 AI 模型和参数

## 4. 核心功能

### 4.1 AI 对话

#### 4.1.1 开始新聊天

1. 点击左侧面板的 "新聊天" 按钮
2. 在输入框中输入您的问题或指令
3. 点击发送按钮或按 Enter 键发送

#### 4.1.2 管理聊天历史

- **查看历史聊天**：在左侧面板点击历史聊天记录
- **重命名聊天**：右键点击聊天记录，选择 "重命名"
- **删除聊天**：右键点击聊天记录，选择 "删除"
- **清空所有聊天**：在设置中选择 "清空所有聊天"

#### 4.1.3 消息操作

- **复制消息**：点击消息气泡右上角的复制图标
- **删除消息**：点击消息气泡右上角的删除图标
- **查看消息详情**：右键点击消息，选择 "查看详情"

### 4.2 文件管理

#### 4.2.1 上传文件

1. 点击右侧面板的 "文件" 标签
2. 点击 "上传文件" 按钮或拖放文件到指定区域
3. 支持的文件格式：PDF、Word、Markdown、TXT 等

#### 4.2.2 创建知识库

1. 上传文件后，点击 "创建知识库"
2. 为知识库命名
3. 选择文本分割策略
4. 点击 "创建" 按钮

#### 4.2.3 管理文件和文件夹

- **查看文件列表**：在右侧面板的文件标签中查看
- **创建文件夹**：点击 "新建文件夹" 按钮
- **移动文件**：拖拽文件到目标文件夹
- **删除文件**：右键点击文件，选择 "删除"

### 4.3 RAG 功能

#### 4.3.1 使用知识库进行对话

1. 确保您已创建知识库
2. 在聊天输入框中输入问题
3. AI 会自动检索相关文档内容并生成回答
4. 查看引用的文档来源

#### 4.3.2 调整 RAG 参数

1. 点击设置按钮
2. 选择 "RAG 设置"
3. 调整以下参数：
   - 检索相似度阈值
   - 最大检索结果数
   - 文本分割大小
   - 重叠比例

### 4.4 MCP 控制面板

#### 4.4.1 切换 AI 模型

1. 点击右侧面板的 "MCP" 标签
2. 在模型列表中选择您想要使用的模型
3. 点击 "应用" 按钮

#### 4.4.2 调整模型参数

1. 在 MCP 面板中选择一个模型
2. 调整模型参数：
   - 温度（temperature）
   - 最大 tokens
   - 上下文窗口大小
   - 其他模型特定参数
3. 点击 "保存" 按钮

#### 4.4.3 管理模型

- **添加模型**：点击 "添加模型" 按钮，输入模型信息
- **编辑模型**：点击模型右侧的编辑图标
- **删除模型**：点击模型右侧的删除图标

## 5. 高级功能

### 5.1 上下文可视化

1. 点击聊天消息右上角的 "查看上下文" 图标
2. 在可视化界面中查看对话上下文关系
3. 调整可视化设置：
   - 布局方式
   - 节点大小
   - 连接样式

### 5.2 知识图谱可视化

1. 点击右侧面板的 "知识图谱" 标签
2. 查看从文档中提取的知识图谱
3. 交互操作：
   - 缩放和平移
   - 点击节点查看详情
   - 调整图谱布局

### 5.3 快捷命令

在输入框中使用以下快捷命令：

- `/new`：开始新聊天
- `/clear`：清空当前聊天
- `/model <model-name>`：切换 AI 模型
- `/help`：显示帮助信息

## 6. 设置与偏好

### 6.1 基本设置

1. 点击顶部导航栏的设置按钮
2. 选择 "基本设置"
3. 调整以下选项：
   - 语言偏好
   - 主题（浅色/深色）
   - 字体大小
   - 通知设置

### 6.2 AI 设置

1. 点击顶部导航栏的设置按钮
2. 选择 "AI 设置"
3. 调整以下选项：
   - 默认 AI 模型
   - 自动保存聊天记录
   - 消息预览长度

### 6.3 RAG 设置

1. 点击顶部导航栏的设置按钮
2. 选择 "RAG 设置"
3. 调整以下选项：
   - 默认检索策略
   - 文档处理设置
   - 向量存储设置

## 7. 故障排除

### 7.1 常见问题

#### 7.1.1 应用无法启动

- 检查系统是否满足要求
- 尝试重新安装应用
- 查看应用日志文件

#### 7.1.2 AI 回答缓慢

- 检查网络连接
- 尝试切换到本地模型
- 调整模型参数（降低温度，减少最大 tokens）

#### 7.1.3 文件上传失败

- 检查文件大小是否超过限制
- 检查文件格式是否支持
- 检查磁盘空间

#### 7.1.4 聊天记录丢失

- 检查是否开启了自动保存
- 查看备份文件
- 联系支持团队

### 7.2 日志与调试

1. 点击顶部导航栏的设置按钮
2. 选择 "高级设置"
3. 开启 "调试模式"
4. 查看日志文件或控制台输出

## 9. 隐私与安全

### 9.1 数据存储

- 聊天记录存储在本地数据库中
- 文档文件存储在本地文件系统
- 向量数据存储在本地向量数据库

### 9.2 数据保护

- 敏感信息加密存储
- 支持数据导出和备份
- 可选择清除所有数据

### 9.3 权限管理

- 应用仅请求必要的系统权限
- 用户可在系统设置中管理应用权限

## 10. 支持与反馈

### 10.1 获取帮助

- **在线文档**：访问官方网站查看详细文档
- **常见问题**：在设置中查看 FAQ
- **联系支持**：通过应用内反馈功能或官方邮箱联系

### 10.2 提供反馈

1. 点击顶部导航栏的设置按钮
2. 选择 "反馈"
3. 填写反馈内容
4. 点击 "提交" 按钮

### 10.3 版本更新

- 应用会自动检查更新
- 您可以在设置中手动检查更新
- 更新日志会显示在更新界面

## 11. 术语表

- **AI**：人工智能（Artificial Intelligence）
- **RAG**：检索增强生成（Retrieval-Augmented Generation）
- **MCP**：模型控制面板（Model Control Panel）
- **Tauri**：跨平台桌面应用开发框架
- **Vue 3**：前端 JavaScript 框架
- **向量数据库**：用于存储和检索向量数据的数据库
- **知识库**：包含结构化和非结构化信息的数据库
